Integration Instructions

To set up Gmail integration:

  1. Enable 2-step verification in your Gmail account
  2. Generate an App Password in Gmail Security settings
  3. Enter your Gmail address and app password above
  4. Save settings to store them securely
  5. Tickets sync server-side every 15 minutes (use “Sync Emails Now” for immediate testing)

Note: App passwords are stored encrypted in the database and are never saved in your browser.
