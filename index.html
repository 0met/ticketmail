Integration Instructions

To set up Gmail integration:

  1. Enable 2-step verification in your Gmail account
  2. Generate an App Password in Gmail Security settings
  3. Enter your Gmail address and app password above
  4. Click "Test Connection" to verify the setup
  5. Save settings and start managing tickets!

Note: This demo uses client-side storage. For production, consider using a secure backend service.
